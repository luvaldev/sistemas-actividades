# Tarea 3: Simulación de Paginación SO

Este proyecto implementa una simulación de gestión de memoria mediante paginación, manejando memoria física (RAM) y memoria virtual (Swap) de la Tarea 3 de Sistemas Operativos.

## Integrante
* Luis Valdenegro Mondaca (Sección 3)

## Compilación y Ejecución
El código está diseñado para ser compilado en un entorno UNIX (Linux/WSL) utilizando el compilador `gcc`.

**1. Compilación:**
Abre una terminal en la carpeta del proyecto y ejecuta el siguiente comando:

```bash
gcc simulador.cp -o simulador
```

**2. Ejecución:**
Para ejecutar el programa, utiliza el siguiente comando en la terminal:

```bash
./simulador
```
